How to Migrate
==============

MrCode stores your settings at:

- __Windows__: `%APPDATA%\MrCode\User`
- __macOS__: `~/Library/Application Support/MrCode/User`
- __Linux__: `~/.config/MrCode/User`

Use the following scripts to migrate:

- [Windows](./windows.md)
- [macOS](./macos.md)
- [Linux](./linux.md)
